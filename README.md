# 💸 Finance Dashboard (React + Vite)

A modern **Finance Dashboard web app** built with **React, Vite, JavaScript, Tailwind CSS, shadcn/ui, Context API, and Recharts**.

This project was created as a **frontend assignment-ready SaaS-style dashboard** with clean UI, smooth UX, responsive layout, dark/light theme, Indian finance mock data, role-based access, and localStorage persistence.
---

## ✨ Features

* 📊 **Interactive Dashboard Overview**

  * Summary cards for income, expenses, savings, and balance
  * Trend charts and category analytics
  * Monthly spending visualizations

* 💳 **Transaction Management**

  * Add new transactions (Admin only)
  * Delete transactions
  * Dynamic category dropdown from existing data
  * Search + filter transactions
  * 100+ Indian mock transaction records

* 🔐 **Role-Based UI (RBAC)**

  * **Viewer:** Can only view data
  * **Admin:** Can add/delete transactions

* 🌗 **Dark / Light Theme**

  * Smooth theme switching
  * Premium SaaS-like UI tokens
  * Theme-aware charts and cards

* 📱 **Fully Responsive Layout**

  * Sticky sidebar for desktop
  * Drawer sidebar for mobile
  * Optimized spacing for all screen sizes

* 💾 **Persistent Data Storage**

  * Uses **localStorage** to save transactions
  * Data remains after page refresh

* 🎨 **Premium UI/UX Polish**

  * Hover animations
  * Smooth transitions
  * Card shadows and rounded layouts
  * Clean spacing hierarchy

# 🛠️ Tech Stack

## Core

* **React.js** → Component-based frontend
* **Vite** → Fast build tool and dev server
* **JavaScript (JSX only)** → Entire project uses `.jsx`

## Styling

* **Tailwind CSS** → Utility-first styling
* **shadcn/ui** → Reusable clean UI components

## State Management

* **Context API** → Global finance state management
* `useContext` + custom hooks for clean state access

## Charts & Data Visualization

* **Recharts**

  * Bar charts
  * Pie charts
  * Line charts

## Storage

* **localStorage** → Persist transactions locally

## Routing

* **React Router DOM**

  * `/dashboard`
  * `/transactions`
  * `/insights`

# 🚀 How to Run Locally

## 1) Clone Repository

```bash
git clone <your-repo-link>
cd finance-dashboard
```

## 2) Install Dependencies

```bash
npm install
```

## 3) Start Development Server

```bash
npm run dev
```

## 4) Open Browser

```bash
http://localhost:5173
```
